## English

**App name (≤30 chars)**
ZIP TO PHOTOLINE

**Short description (≤80 chars)**
Extract photos from ZIP files and share straight to LINE — no manual saving.

**Full description (≤4000 chars)**

Got a ZIP file full of photos from a friend, cloud drive, or work email — and don't want to extract it manually before sending to LINE?

ZIP TO PHOTOLINE lets you:

📦 Auto-extract ZIP files — just share a ZIP file to this app from any app (LINE Files, Drive, email, etc.) and it instantly extracts every image inside.

🖼️ Familiar chat-style selection — tap to select photos, with numbered order badges. Select up to 50 photos at once. Long-press to view full screen.

🔗 Share instantly — one tap sends your selected photos to LINE or any other app, no need to save to your gallery first.

🔒 100% private — everything runs on-device. No internet connection, no data collection, no ads.

Supported image formats: JPG, PNG, GIF, WEBP, BMP, HEIC/HEIF

Perfect for quickly forwarding event photos, work files, or shared albums that arrive as a ZIP into a LINE group chat.

## หมายเหตุก่อนอัปโหลด (Notes before submitting)

1. **Privacy Policy URL** — ต้องนำไฟล์ `privacy-policy.html` ในโฟลเดอร์นี้ไปโพสต์ออนไลน์ก่อน (เช่น GitHub Pages, Google Sites ฟรี) แล้วนำ URL ไปกรอกใน Play Console > App content > Privacy Policy
2. **Screenshots** — Play Console ต้องการสกรีนช็อตอย่างน้อย 2 รูป (แนะนำ 4-8 รูป) ขนาดมือถือ ผมมี screenshot จาก emulator ที่ทดสอบไว้แล้ว ถ้าต้องการใช้เป็นชุดทางการสามารถถ่ายใหม่แบบมีรูปภาพตัวอย่างให้ดูสวยงามได้
3. **Feature graphic** — ต้องมีรูปขนาด 1024x500px สำหรับหน้า Store listing (ยังไม่มี ต้องออกแบบเพิ่ม)
4. **Data safety form** — ใน Play Console ต้องตอบแบบฟอร์ม "Data safety" ตามนโยบายนี้: เลือก "No data collected" เพราะแอปไม่เก็บข้อมูลใดๆ
5. **Content rating questionnaire** — ตอบตามจริงว่าแอปไม่มีเนื้อหารุนแรง/ผู้ใหญ่ ฯลฯ จะได้เรต PEGI 3 / Everyone
